# [HW14] 문자열 생략하여 출력하기

<h3>
문자열을 입력 받아 입력 받은 문자열을 앞에서부터 반만 출력하시오.</br>
출력 시 대괄호[]로 묶어서 출력하되 문자열의 길이만큼의 필드폭에 문자열의 반만 출력하고 문자열 뒤에는 생략기호(...)를 붙여서 출력하시오.
</br></br>
(실행결과)
</br></br></h3>

```
* 문자열 입력 : stream
[□□□str. . .]
* 문자열 입력 : dream
[□□□dr. . .]
* 문자열 입력 : at
[□a. . .]
* 문자열 입력 : a
[□. . .]
```
