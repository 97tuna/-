# [HW36] 피보나치 수열 구하기


<h4>

1항부터 제 N항까지 피보나치 수열의 합을 계산하여 출력하는 프로그램 작성
피보나치 수열은 첫 번째 항과 두 번째 항을 더해서 세 번째 항을 만들고, 두 번째 항과 세 번째 항을 더해서 네 번째 항을 만드는 방법으로 계속해서 다음 항을 만들어가는 수열이다.
